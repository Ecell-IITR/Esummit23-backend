## API Reference

#### Get all items

```http
  GET http://127.0.0.1:8000/design/colors/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `NONE` | `string` | **Required**. NA |

#### Get item

```http
  GET http://127.0.0.1:8000/public/team
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `NONE`      | `string` | **Required**. NA |
